| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-01 | Tampering | TikTok hashtag slug | mitigate | Slug strips all non-alphanumeric via `replace(/[^a-z0-9]/g,'')` so injected markup/control chars cannot survive into hashtags |
| T-02-02 | Denial of Service | processor.generate throwing | mitigate | buildFallback is a non-throwing deterministic twin; 02-02 fan-out uses it on rejection so one processor cannot block others (feeds PIPE-04) |
